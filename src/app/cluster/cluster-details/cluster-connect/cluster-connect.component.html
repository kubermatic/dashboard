Configure kubectl by running the following command in the directory where the kubeconfig-admin-{{cluster.id}} is located:

$ export KUBECONFIG=$PWD/kubeconfig-admin-{{cluster.id}}

Then start a proxy to connect to the Kubernetes control plane:

$ kubectl proxy

Then open the Kubernetes Dashboard by using following link in your browser:

http://localhost:8001/api/v1/namespaces/kube-system/services/https:kubernetes-dashboard:/proxy/
