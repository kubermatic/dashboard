Security Group ID When specified, all worker nodes will be attached to this security group. If not specified, a security group will be created. The security group ID need to look similar to 'sg-a2c195ca'. Route Table ID When specified, all worker nodes will be attached to this Route table. If not specified, the default Route Table will be used. The route table ID needs to look similar to 'rtb-0bd18f60'. Instance Profile Name When specified, the worker nodes will use this instance profile. If not specified, a new instance profile will be created. Control plane role (ARN) When specified, the Kubernetes control plane will use this role when communicating with the AWS API. If not specified, a new role will be created.
